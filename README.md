# TrickyMath - a new version of Katemath


## How to run:

  ```
git clone git@github.com:CezarySzukiel/KateProject.git
cd KateProject/backend_core/env/
mv .env_demo .env
docker compose up -d
```

because docker frontend image is temporary disabled: 
```
cd ../../frontend
npm install
npm run dev
```
